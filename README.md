# Matplotlib Practice-Projects

## Overview

This repository contains multiple mini-projects built while learning data visualization using Python, Pandas, and Matplotlib.

Each project is organized in its own folder and includes:

* Dataset
* Complete analysis code
* Project-specific README with insights

The focus is on building practical visualization skills and developing the ability to extract meaningful insights from data.

## How to Use

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Install dependencies:

```bash
pip install pandas matplotlib
```

3. Navigate to any project folder:

```bash
cd student-performance-analysis
```

4. Run the script:

```bash
python analysis.py
```
